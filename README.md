# Noter

მსუბუქი, გარე ბიბლიოთეკებისგან დამოუკიდებელი (zero-dependency) ჩანაწერების აპლიკაცია. შთაგონებულია **i3wm**-ის ფანჯრული განლაგებითა და **Obsidian**-ის მარკდაუნ (Markdown) პრინციპებით. მუშაობს სრულიად ოფლაინ რეჟიმში, ერთ სუფთა HTML ფაილში.

## ✨ ფუნქციები

*   **i3wm-ის სტილის განლაგება (Tiling Layout):** ავტომატურად ითვისებს მთელ ეკრანს და არ ტოვებს ცარიელ ადგილს. ფანჯრები შეგიძლიათ გაყოთ ვერტიკალურად (`◫`) ან ჰორიზონტალურად (`⊟`).
*   **ზომის შეცვლა შუიდან:** ფანჯრებს შორის არსებული გამყოფების მაუსით გადაადგილებით მარტივად დაარეგულირეთ თითოეული ბლოკის პროპორცია.
*   **Obsidian-ის სტილის მარკდაუნი:** წერეთ ჩვეულებრივი ტექსტი, ხოლო ფანჯრიდან მაუსის გატანით ან `Esc` ღილაკზე დაჭერით მომენტალურად დაარენდერეთ ჰედერები, ხაზები (`---`), მუქი ტექსტი და კოდის ბლოკები. ჩასასწორებლად უბრალოდ ხელახლა დააკლიკეთ ტექსტს.
*   **ლოკალური შენახვა (Cache):** ტექსტიც და ფანჯრების განლაგების სტრუქტურაც ავტომატურად ინახება ბრაუზერის ლოკალურ მეხსიერებაში (`Local Storage`).
*   **უსაფრთხოების ჩაკეტვა:** ბოქლომის ხატულაზე (`🔓`/`🔒`) დაჭერით შეგიძლიათ ჩაკეტოთ სასურველი ფანჯარა, რაც დაიცავს მის სათაურსა და ტექსტს შემთხვევითი წაშლის ან რედაქტირებისგან.
*   **მონაცემების პორტატულობა:** ჩანაწერების სრული ბექაფი ექსპორტდება ფაილად სახელით `noti-YYYY-MM-DD.json`, რომლის იმპორტიც ნებისმიერ დროს შეგიძლიათ ხელახლა.
*   **7 ჩაშენებული თემა:** Dracula (ნაგულისხმევი), Light, Dark, Sepia, Matrix, Gruvbox და Nord.
*   **100%-ით პრივატული:** კოდი არ იყენებს გარე სკრიპტებს და მუშაობს ინტერნეტის გარეშეც.

## ⌨️ კლავიატურის კომბინაციები

*   `Ctrl + B` : მონიშნული ტექსტის **გამუქება** (Bold).
*   `Esc` : ტექსტის რედაქტირებიდან მარკდაუნის (Markdown) ხედვაზე მომენტალურად გადასვლა.

## 🚀 როგორ გავუშვათ

1. შეინახეთ პროექტის კოდი `index.html` ფაილად.
2. ორჯერ დააკლიკეთ ფაილს და გახსენით ის ნებისმიერ ვებ-ბრაუზერში.
